# EyeBreak v1.0 — Initial Release 🎉

## What is EyeBreak?

A macOS menu bar app that reminds you to rest your eyes using the **20-20-20 rule**. When it's break time, your screen fills with TV static — so boring you naturally look away.

## Highlights

- 👁️ Smart 20-20-20 reminders
- 📺 TV static overlay (anti-attention design)
- ⏱️ Menu bar popover with arc progress timer
- ⏸️ Full timer controls: pause, resume, snooze, skip
- 🌍 Arabic + English with auto-detection
- 🔄 Built-in auto-update checker
- 🪶 Only ~100KB, zero dependencies

## Install

1. Download **EyeBreak-1.0.dmg** below
2. Drag to `/Applications`
3. Right-click → Open (first time only, due to macOS security)

## Requirements

- macOS 13 (Ventura) or later
